About the project

The topic of our project is private clinic promo site

Goals

Goal of this project is to create a site for private clinic;


User

Our customer is head of the private clinic's personnel department .

The site users are potential clients of this clinic.



Functional requirements

-The site should contain major information about clinic

-The site should be able to interact with client

-The site should be able to contain information about reception of patients

Non-​Functional requirements

-The site should be easy-to use for not advanced users

-The site should work properly on mobile devices
